Personal AI Projects

  1. GPT Otimization with RAG - Optimized GPT query responses on data by implementing Retrieval Augmented Generation, which improved accuracy and reliability of answers.
  2. SpotOn - Review analysis app that fetches place details using Google Places API and answers queries based on reviews from user-selected locations. Displays a user-friendly interface with Google Maps for seamless exploration, search, and interaction.
  3. aiXtensions - Created two Google Chrome extensions, [LookItUp] for defining text and providing synonyms/antonyms, and [Persona] for transforming text into different styles of language.
  4. CooPA - Personal assistant app that answers queries by dynamically web scraping relevant articles from Google Search. Features speech recognition and TTS for an enhanced user experience.
  5. Qdoc - Web app that summarizes articles (URLs/PDFs) and answers queries about article content. Features a conversation history for context-aware responses.
