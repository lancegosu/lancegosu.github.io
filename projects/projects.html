Personal AI Projects

My goal in creating AI projects is to harness the transformative capabilities of Large Language Models (LLMs) and seamlessly integrate them into diverse applications. By leveraging the power of LLMs, I aim to enhance the intelligence, functionality, and user experience of these applications, pushing the boundaries of what AI can achieve in various domains. This endeavor reflects my commitment to unlocking the full potential of artificial intelligence to drive innovation and elevate the impact of technology in our daily lives.

  • SpotOn is a Django web application that seamlessly integrates the Google Places API and OpenAI's GPT-3.5-turbo language model. This fusion results in a sophisticated platform for place search and intelligent question-answering.
    • Place Search: Utilize the Google Places API to perform dynamic searches for places based on sophisticated queries, ensuring accurate and comprehensive results.
    • Review-based Q&A: Harnessing the power of OpenAI's GPT-3.5-turbo language model, SpotOn dynamically generates coherent and contextually relevant answers to user queries. This approach utilizes selected place reviews, creating a robust and adaptive conversational interface.
    • Interactive Map: Enhance user experience by visually representing search results on an interactive map, providing users with a spatial understanding of their queries.
    • Dynamic UI: The user interface dynamically updates, enabling users to seamlessly ask questions about selected places and receive AI-generated responses in real-time.
    • Libraries used: Django, Google Places API, Google Maps Api, OpenAI GPT-3.5-turbo API
  • CooPA
    • Personal assistant app that answers queries by dynamically web scraping relevant articles from Google Search.
    • Features speech recognition and TTS for an enhanced user experience.
  • Qdoc
    • Web app that summarizes articles (URLs/PDFs) and answers queries about article content.
    • Features a conversation history for context-aware responses.
  • GPT Optimization with RAG
    • Optimized GPT query responses on data by implementing Retrieval Augmented Generation, improving accuracy and reliability of answers.
  • aiXtensions
    • Created two Google Chrome extensions, LookItUp for defining highlighted text and providing synonyms/antonyms, and Persona for transforming highlighted text into different styles of language.
